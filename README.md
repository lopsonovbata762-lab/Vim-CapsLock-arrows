Vim j/k/l/i arrows navigation for Windows, anywhere

Vim-style Windows Navigation (AHK v2)
Simple AutoHotkey v2 script that turns your CapsLock into a navigation layer while keeping
its original toggle function.

How it works:

. Tap CapsLock: Toggles standard CapsLock on/off.

. Hold CapsLock: Activates the navigation layer.

· I, J, K, L: Up, Left, Down, Right (Arrow keys).

· U, O: Home, End (Line navigation).

. S (Modifier): Acts as Ctrl while in navigation mode (jump by words).

· D (Modifier): Acts as Shift while in navigation mode (select text).

. S + D: Combine both to select text word-by-word.

Installation:

1. Install AutoHotkey v2 from autohotkey.com

2. Download Vim CapsLock arrows.ahk

3. Press Win + R, type shell: startup , and hit Enter.

4. Move Vim CapsLock arrows. ahk into the folder that opened (this ensures the script starts
with Windows).

5. Run the script (double-click the file).
